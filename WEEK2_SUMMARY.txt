┌──────────────────────────────────────────────────────────────────────────────┐
│ BACKEND ENDPOINTS (routes.py)                                               │
└──────────────────────────────────────────────────────────────────────────────┘

✅ GET  /api/weather/{spot_id}?day_index=0
   → Live multi-source weather with Para-Index scoring
   → Returns consensus data from 5 weather sources
   → Includes hourly slots and flyability verdict

✅ GET  /api/flights?limit=10
   → Recent flights from database
   → Ordered by date (newest first)

✅ GET  /api/flights/stats
   → Aggregate statistics (total flights/hours/distance)
   → Average duration, favorite spot, last flight date

✅ GET  /api/flights/{flight_id}/gpx-data
   → Parsed GPX coordinates for Cesium 3D rendering
   → Elevation profile, distance, duration calculated
   → Returns array of {lat, lon, elevation, timestamp}

✅ GET  /api/flights/{flight_id}/gpx
   → Download original GPX file as attachment

✅ GET  /api/alerts
   → Dynamic weather alerts based on conditions
   → Filters by spot_id (optional)

✅ POST /api/alerts
   → Create custom alerts (placeholder for Phase 4)
